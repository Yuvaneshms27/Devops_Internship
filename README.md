
Overview

This project demonstrates the implementation of a basic DevOps workflow including source code analysis and quality checks using SonarQube integration.

Tasks Completed
	•	Configured a CI/CD workflow using GitHub Actions
	•	Integrated SonarQube for static code analysis
	•	Executed automated code scanning during pipeline runs
	•	Ensured all quality gates passed successfully (no major vulnerabilities or bugs detected)
	•	Validated project build and scan results in the SonarQube dashboard

Tools & Technologies Used
	•	Git & GitHub
	•	GitHub Actions
	•	SonarQube
	•	Docker 

Outcome

The project pipeline runs successfully with all checks passing, ensuring clean, maintainable, and secure code standards.
